## 解答
```mermaid
flowchart TB
  START-->A(購入者が現れる)
  A(購入者が現れる)-->B{お金を入れたか}
  B--Yes-->C(金額を表示)
  B--No-->START
  D3-->START
  C-->返却ボタンの処理-->E{購入ボタンが押されたか}
  D1--No-->E
      subgraph 返却ボタンの処理
    D1{返却ボタンを押されたか}--Yes-->D2(お金を返却する)
    D2-->D3(金額の表示をリセットする)
    end

  E{購入ボタンが押されたか}--Yes-->F{指定以上の金額が入っているか}
  E--No-->B
  F--Yes-->G(飲み物を出す)
  F--No-->B
  G-->END
```
